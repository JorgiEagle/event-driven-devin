# event-driven-devin

Devin Automation for Superset
This project automates maintenance tasks in any github repository using Devin.
When an issue is created or labeled, the system triggers a Devin session that fixes the issue and opens a pull request.

## Features
Github issue listing and dashboard
Manual task selection and kick off
Togglable Event‑driven issue remediation

## Outputs
Automatic PR creation
Lightweight observability
Logging
Status updates for each task
